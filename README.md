# Sixth Mass Extinction: Temporal Insurgency

> "You can't change the past. But you can create a better future from it."

## What Is This?

**Sixth Mass Extinction** is an open-source, community-driven video game project. You play as a scientist from 2055 who travels back to 2025 to prevent human extinction before 2035. It's a survival shooter with revolutionary AI, cybersecurity puzzles, temporal paradoxes, and sanity mechanics.
